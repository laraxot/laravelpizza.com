# Docs Root Location Skills

## Skill 1: Global Docs Consolidation
1. Sposta contenuti da `laravel/docs/` a `docs/`.
2. Verifica assenza directory sorgente.
3. Esegui check su file critici post-migrazione.

## Skill 2: Relative Link Rebase
1. Cerca link verso `docs/` nei moduli/temi.
2. Aggiungi un livello `../` per riallineare i percorsi al root.
3. Riesegui validazione link.

## Skill 3: Governance Update
1. Aggiorna files in `docs/rules`, `docs/memory`, `docs/architecture`.
2. Collega le nuove regole dagli indici principali del modulo/tema impattato.
