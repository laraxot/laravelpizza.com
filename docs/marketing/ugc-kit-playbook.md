# UGC Kit Playbook (Sprint 1)

## Obiettivo
Aumentare UGC entro 72h dopo ogni pizzata/meetup.

## Kit base

1. Hashtag ufficiale evento + hashtag brand.
2. 3 prompt post:
   - "cosa hai imparato"
   - "chi hai conosciuto"
   - "il tuo take-away tecnico"
3. Template caption breve (IT/EN).
4. CTA finale: invita un amico al prossimo evento.

## Sequenza operativa

- T-24h: invio reminder con kit UGC.
- T+2h: push post-event con prompt #1.
- T+24h: follow-up con prompt #2.
- T+48h: follow-up con prompt #3 e CTA referral.

## KPI

- UGC posts 72h
- UGC creators unici
- Reach stimata dei post community

## Qualita'

- no spam
- tono inclusivo
- no contenuti offensivi o non pertinenti
