# Viral Sprint Backlog 2026 Q2

## Sprint 1 (2 settimane) - Tracking + UGC Foundation

### Obiettivo
Attivare il loop Event -> UGC -> Discovery con tracking affidabile.

### Deliverable
1. KPI dashboard base (RSVP, show-up, UGC within 72h, referral invites).
2. UGC kit ufficiale (template post, hashtag, brief speaker/attendee).
3. Post-event automation: recap + CTA referral.

### KPI target sprint
- +20% UGC rate sugli eventi pilota.
- baseline completa su RSVP->show-up.

## Sprint 2 (2 settimane) - Referral Loop

### Obiettivo
Attivare Event -> Referral -> RSVP.

### Deliverable
1. meccanica referral (invite link/codice).
2. reward schema doppio lato.
3. reminder cadence T-48h / T-6h / T-1h.

### KPI target sprint
- +15% RSVP da referral.
- no-show rate in riduzione rispetto baseline.

## Sprint 3 (2 settimane) - Retention + Host Scale

### Obiettivo
Attivare Post-event -> Retention -> New Hosts.

### Deliverable
1. NPS + micro-survey post evento.
2. Ambassador/host candidate funnel.
3. playbook host locale v1.

### KPI target sprint
- repeat attendance 30d in crescita.
- almeno 3 candidati host attivi.

## Cadence operativa

- Weekly review in GitHub issue con numeri.
- Ogni sprint chiude con retro + decisione go/no-go su esperimenti.
