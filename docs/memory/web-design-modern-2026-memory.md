## Scoperta Chiave
Il web design nel 2026 si basa su 3 pilastri:
1. **Immersività**: L'utente non legge, esplora
2. **Performance**: Core Web Vitals come requisito non negoziabile
3. **Accessibilità**: Requisito legale e non solo etico

## Trend Principali
- Kinetic web design (movimento come elemento centrale)
- Micro-interazioni (feedback immediati)
- Scrollytelling (narrativa attraverso scroll)
- Mobile-first reale (non solo responsive)
- Dark mode
- Design emotivo

## Metriche Core Web Vitals
- LCP < 2.5s
- CLS < 0.1
- INP < 200ms

## Best Practices Animazioni
- Preferire CSS transforms/opacity (GPU)
- Durata massima 300ms per micro-interazioni
- Rispetta prefers-reduced-motion
- Coerenza nel timing e easing

## Accessibilità WCAG
- Contrasti 4.5:1 minimo
- Navigazione tastiera
- ARIA labels
- Skip links
- Focus visibile

## Micro-interazioni
1. Animazioni hover su pulsanti
2. Feedback moduli (validazione)
3. Transizioni pagine
4. Loader/cursori
5. Contatori real-time
6. Scroll direzionale
