# Livewire Volt Filament Stack Memory

Memoria operativa:

- Folio decide la route, Volt/Livewire decide il comportamento interattivo della pagina, Filament 5 decide il panel;
- se una pagina frontoffice e' interattiva, prima domanda: puo' essere una page Folio con Volt?;
- se una UI appartiene all'admin/panel, prima domanda: puo' essere espressa come Resource/Page/Widget/Schema/Action di Filament?;
- il plugin Google Fonts e' panel-scoped, non theme-scoped.
