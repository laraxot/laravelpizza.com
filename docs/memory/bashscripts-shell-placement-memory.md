# Bashscripts Shell Placement Memory

## Lezione appresa

Gli script shell devono vivere sotto `bashscripts/` e non sotto `laravel/`.

## Standard operativo

- Collocare `.sh` in `bashscripts/scripts/*` o sottocartella coerente.
- Tenere la documentazione aggiornata in `bashscripts/docs/`.
- Tracciare ogni spostamento in GitHub Issues e Discussions.
