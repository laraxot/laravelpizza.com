# Meetup GDPR Profile E2E Memory

## Sintesi

Il progetto richiede un journey completo stile social/community:
- proposta meetup;
- moderazione/approvazione;
- visibilita' differenziata pending/public;
- RSVP con contatori;
- commenti;
- GDPR completo (consensi obbligatori + opzionali aggiornabili);
- profilo/password utente.

## Stato

Base tecnica presente ma manca chiusura di alcuni flussi cross-modulo e relativi test E2E Pest.

## Direzione

Procedere per tranche modulo-per-modulo, aggiornando issue/discussion a ogni step.
