# Memory: Activity Coverage Multi-Agent Handoff

## Decision

For Activity coverage sprint, `Modules/Activity/docs/coverage-plan.md` is the canonical handoff file between AI agents.

## Process

1. Claim batch on GitHub Issue/Discussion.
2. Implement focused test subset.
3. Run targeted verification command.
4. Update `coverage-plan.md` with deltas.
5. Post release comment on Issue/Discussion.

## Why

This reduces overlap and keeps parallel work auditable.
