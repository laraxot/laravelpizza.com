# Viral Growth Loop Skill

## Goal

Tradurre obiettivi di crescita in loop misurabili e sprint operativi.

## Workflow

1. Selezionare loop principale (UGC/referral/content/retention).
2. Definire KPI + baseline + target.
3. Pianificare 2-5 esperimenti A/B.
4. Eseguire sprint 2 settimane.
5. Pubblicare risultati su GitHub issue/discussion.
