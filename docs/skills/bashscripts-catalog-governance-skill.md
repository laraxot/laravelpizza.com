# Bashscripts Catalog Governance Skill

## Goal

Garantire che tutti gli script shell siano nel posto corretto e documentati.

## Workflow

1. Cercare script shell fuori da `bashscripts/`.
2. Spostarli in `bashscripts/scripts/*`.
3. Aggiornare riferimenti nei comandi/docs.
4. Aggiornare catalogo in `bashscripts/docs/`.
5. Pubblicare avanzamento su issue/discussion.
