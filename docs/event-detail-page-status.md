# Event Detail Page - Current State

## Status: 17 Feb 2026

### Note
The original laravelpizza.com site is currently showing a "Hostinger Horizons" error page, making it impossible to compare with the original design. The current implementation is based on the legacy documentation and existing codebase patterns.

## Current Implementation

### URL Pattern
- **Locale-aware**: `/it/events/{slug}`, `/en/events/{slug}`, etc.
- **Routing**: Handled by `Themes/Meetup/resources/views/pages/[slug].blade.php`
- **Event lookup**: Via `Event::where('slug', $eventSlug)->first()`

### Page Components

#### 1. Navigation Header
- Logo with link to home
- Navigation links: Home, Events, Community Chat
- Language selector dropdown (6 languages)

#### 2. Breadcrumb
```
Events / Laravel 11 Release Pizza Party
```

#### 3. Event Header (Card)
- **Cover Image**: Placeholder with calendar icon OR event image if available
- **Status Badge**: "Upcoming" (green) or "Past Event" (gray)
- **Title**: Event title (h1, 3xl-4xl)

#### 4. Event Details Grid (2 columns)
- **Date**: Day name, Month day, Year (e.g., "Monday, February 24, 2026")
- **Time**: Start - End (e.g., "6:01 PM - 9:01 PM")
- **Location**: Venue address
- **Language**: (if set) e.g., "EN"
- **Attendees**: "X / Y spots filled"

#### 5. Description Section
- **Title**: "About this event"
- **Content**: Event description with preserved line breaks

#### 6. CTA Button
- "Register Now" button (red, appears only for upcoming events)

#### 7. Footer
- "Back to all events" link

#### 8. SEO
- Schema.org JSON-LD structured data (via `$event->toSchemaOrg()`)

## Fix Applied (17 Feb 2026)

### Issue
Event links were generating `/events/{slug}` instead of `/it/events/{slug}` (or appropriate locale prefix).

### Solution
Changed in `list.blade.php`:
```blade
{{-- OLD (broken) --}}
<a :href="'{{ LaravelLocalization::localizeUrl('/events') }}/' + event.slug">

{{-- NEW (fixed) --}}
<a :href="event.url">
```

The `event.url` is now properly generated by `Event::toBlockArray()` using:
```php
'url' => LaravelLocalization::localizeUrl('/events/'.$this->slug),
```
