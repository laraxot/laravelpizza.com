# Viral Marketing Experimentation Rule

## Regola

Il marketing virale deve essere gestito come sistema sperimentale misurabile, non come attivita' spot.

## Obblighi

1. Ogni iniziativa deve dichiarare loop target (UGC/referral/retention/discovery).
2. Ogni esperimento deve avere KPI, baseline e finestra temporale.
3. Ogni sprint marketing deve aggiornare GitHub Issue e Discussion con risultati.
4. Nessun claim di "viralita'" senza evidenza dati.
