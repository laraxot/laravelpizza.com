# MCP (Model Context Protocol) Integration in Themes

## Overview
The Laravel Pizza project includes MCP (Model Context Protocol) integration in the Themes/Meetup, providing AI-assisted development capabilities.

## MCP Configuration

### Cursor MCP Configuration
The `.cursor/mcp.json` file defines the available MCP servers:
- Filesystem server for file operations
- Git server for version control operations
- Database server for database operations
- Artisan server for Laravel command execution
- PHPStan server for static analysis
- Composer server for dependency management

### MCP Capabilities
- File system operations (read, write, list, etc.)
- Git operations (status, commit, push, etc.)
- Database operations (queries, migrations, etc.)
- Laravel Artisan commands execution
- Code quality tools integration
- Dependency management
