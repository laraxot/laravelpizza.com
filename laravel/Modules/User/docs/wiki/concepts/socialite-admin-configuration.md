# Socialite Admin Configuration

## The Goal

Allow administrators to configure OAuth provider credentials (`GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`, etc.) via Filament backoffice UI, **without editing .env files or redeploying**.

---

## The Problem

**Current State:**
```bash
# .env file (requires deploy to change)
GOOGLE_CLIENT_ID=xxx.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=GOCSPX-xxx
```

**Problems:**
1. Non-technical admins can't edit .env
2. Each change requires redeploy
3. Secrets visible in environment files
4. No UI for enable/disable providers

---

**Filament Admin Settings Page** → Edit credentials → Save to secure storage → Real-time apply

## Implementation Strategy

### Architecture Decision: Config File Generation

**Chosen Approach**: Admin edits → Generate PHP config → Cache → Real-time apply

```php
// Storage paths
laravel/
├── config/
│   └── services.php              ← Core Laravel config
└── bootstrap/cache/
    └── socialite-providers.php ← Generated by admin (gitignored)
```

**Why not database?**
- Config needed at boot time (before DB connection)
- Socialite reads from `config('services.google')`
- File generation is simpler and more reliable

**Why not env?**
- Non-technical admins
- Requires redeploy
- No versioning issues

## Technical Implementation

### 1. Settings Page (Filament v5)

```php
<?php

namespace Modules\User\Filament\Pages;

use Filament\Forms\Components\Section;
use Filament\Forms\Components\TextInput;
use Filament\Forms\Components\Toggle;
use Filament\Forms\Components\TagsInput;
use Filament\Forms\Form;
use Filament\Pages\Page;
use Illuminate\Support\Facades\Artisan;
use Illuminate\Support\Facades\File;

/**
 * Socialite OAuth Provider Settings Page.
 * 
 * Allows admins to configure Google, GitHub, etc. without editing .env
 */
class SocialiteProviderSettingsPage extends Page
{
    protected static string $view = 'user::filament.pages.socialite-settings';
    protected static ?string $navigationIcon = 'heroicon-o-key';
    protected static ?string $navigationGroup = 'User Management';
    protected static ?int $navigationSort = 20;

    public ?array $google = [];
    public ?array $github = [];
    public ?array $microsoft = [];

    public function mount(): void
    {
        // Load current config
        $this->google = [
            'enabled' => config('services.google.enabled', false),
            'client_id' => config('services.google.client_id', ''),
            'client_secret' => $this->maskSecret(config('services.google.client_secret')),
            'scopes' => config('services.google.scopes', ['openid', 'email', 'profile']),
        ];
        
        $this->github = [
            'enabled' => config('services.github.enabled', false),
            'client_id' => config('services.github.client_id', ''),
            'client_secret' => $this->maskSecret(config('services.github.client_secret')),
        ];
        
        // ... other providers
    }

    public function form(Form $form): Form
    {
        return $form
            ->schema([
                Section::make('Google OAuth')
                    ->icon('ui-google')
                    ->collapsible()
                    ->schema([
                        Toggle::make('google.enabled')
                            ->label('Attiva login con Google'),
                        
                        TextInput::make('google.client_id')
                            ->label('Client ID')
                            ->placeholder('xxx.apps.googleusercontent.com')
                            ->visible(fn ($get) => $get('google.enabled')),
                        
                        TextInput::make('google.client_secret')
                            ->label('Client Secret')
                            ->password()
                            ->revealable()
                            ->placeholder('GOCSPX-xxx')
                            ->dehydrateStateUsing(fn ($state) => 
                                $this->isMasked($state) 
                                    ? config('services.google.client_secret')
                                    : $state
                            )
                            ->visible(fn ($get) => $get('google.enabled')),
                        
                        TagsInput::make('google.scopes')
                            ->label('Scopes')
                            ->default(['openid', 'email', 'profile'])
                            ->visible(fn ($get) => $get('google.enabled')),
                        
                        TextInput::make('google.redirect')
                            ->label('Redirect URL (read-only)')
                            ->default(fn () => route('socialite.callback', 'google'))
                            ->disabled(),
                    ]),
                
                Section::make('GitHub OAuth')
                    ->icon('fab-github')
                    ->collapsible()
                    ->schema([
                        // Similar structure...
                    ]),
            ])
            ->statePath('data');
    }

    public function save(): void
    {
        $data = $this->form->getState();
        
        // Generate config array
        $config = [
            'google' => [
                'enabled' => $data['google']['enabled'] ?? false,
                'client_id' => $data['google']['client_id'] ?? '',
                'client_secret' => $data['google']['client_secret'] ?? '',
                'redirect' => route('socialite.callback', 'google'),
                'scopes' => $data['google']['scopes'] ?? ['openid', 'email', 'profile'],
            ],
            'github' => [
                'enabled' => $data['github']['enabled'] ?? false,
                'client_id' => $data['github']['client_id'] ?? '',
                'client_secret' => $data['github']['client_secret'] ?? '',
                'redirect' => route('socialite.callback', 'github'),
            ],
        ];
        
        // Write to secure config file
        $this->writeSocialiteConfig($config);
        
        // Clear config cache
        Artisan::call('config:clear');
        
        // Update SocialProvider Sushi model data
        $this->updateSocialProviderActiveStates($config);
        
        $this->notify('success', 'Configurazione OAuth salvata con successo');
    }

    /**
     * Write config to secure PHP file (outside git).
     */
    private function writeSocialiteConfig(array $config): void
    {
        $path = storage_path('app/private/socialite-config.php');
        
        // Ensure directory exists
        File::ensureDirectoryExists(dirname($path), 0750, true);
        
        // Generate PHP file content
        $content = "<?php\n\nreturn " . var_export($config, true) . ";\n";
        
        // Write with restricted permissions
        File::put($path, $content);
        chmod($path, 0640);
    }

    /**
     * Mask secret for display (show only last 4 chars).
     */
    private function maskSecret(?string $secret): string
    {
        if (empty($secret)) {
            return '';
        }
        
        return str_repeat('•', strlen($secret) - 4) . substr($secret, -4);
    }

    /**
     * Check if value is masked placeholder.
     */
    private function isMasked(string $value): bool
    {
        return str_contains($value, '•');
    }

    /**
     * Update SocialProvider Sushi model active states.
     */
    private function updateSocialProviderActiveStates(array $config): void
    {
        foreach ($config as $provider => $settings) {
            // Update SocialProvider model active flag
            SocialProvider::updateOrCreate(
                ['name' => $provider],
                ['active' => $settings['enabled'] ?? false]
            );
        }
    }
}
```

### 2. Config Loading (Service Provider)

```php
<?php

namespace Modules\User\Providers;

use Illuminate\Support\Facades\Config;
use Illuminate\Support\ServiceProvider;

/**
 * Merge admin-configured OAuth settings with Laravel config.
 */
class SocialiteConfigServiceProvider extends ServiceProvider
{
    public function register(): void
    {
        // Load admin-generated config
        $adminConfigPath = storage_path('app/private/socialite-config.php');
        
        if (file_exists($adminConfigPath)) {
            $adminConfig = require $adminConfigPath;
            
            foreach ($adminConfig as $provider => $settings) {
                // Merge with existing services config
                Config::set("services.{$provider}", array_merge(
                    Config::get("services.{$provider}", []),
                    $settings
                ));
            }
        }
    }

    public function boot(): void
    {
        // No boot actions needed
    }
}
```

## Security Measures

### 1. File Permissions

```php
// After writing config file
chmod($path, 0640);  // Owner read/write, group read, others none
chown($path, 'www-data');
chgrp($path, 'www-data');
```

### 2. Gitignore

```bash
# .gitignore (project root)
storage/app/private/socialite-config.php
```

### 3. Encryption Option (Advanced)

```php
// Encrypt config before saving
$encrypted = encrypt($config);
File::put($path, $encrypted);

// Decrypt when loading
$encrypted = File::get($path);
$config = decrypt($encrypted);
```

### 4. Audit Logging

```php
// Log all config changes
Activity::create([
    'log_name' => 'socialite',
    'description' => 'OAuth config updated',
    'subject_type' => SocialProvider::class,
    'causer_type' => User::class,
    'causer_id' => auth()->id(),
    'properties' => ['provider' => 'google', 'fields' => ['client_id', 'scopes']],
]);
```

## User Tutorial: Setting up Google OAuth

### Step 1: Create Google Cloud Project

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create new project: "FixCity OAuth"
3. Enable **Google+ API** or **People API**

### Step 2: Configure OAuth Consent Screen

1. APIs & Services → OAuth consent screen
2. User Type: **External**
3. App name: "FixCity"
4. User support email: support@fixcity.local
5. Developer contact: dev@fixcity.local
6. Scopes: `openid`, `email`, `profile`
7. Test users: Add your email

### Step 3: Create Credentials

1. APIs & Services → Credentials
2. Create Credentials → OAuth client ID
3. Application type: **Web application**
4. Name: "FixCity Web"
5. Authorized JavaScript origins: `https://fixcity.local`
6. Authorized redirect URIs: `https://fixcity.local/auth/google/callback`
7. Click **Create**
8. Copy **Client ID** and **Client Secret**

### Step 4: Configure in FixCity Admin

1. Login to FixCity admin: `/admin`
2. Navigate to: **User Management → OAuth Providers**
3. Find **Google OAuth** section
4. Enable: **☑ Attiva login con Google**
5. Paste **Client ID**: `xxx.apps.googleusercontent.com`
6. Paste **Client Secret**: `GOCSPX-xxx`
7. Scopes: `openid`, `email`, `profile`
8. Click **💾 Salva Configurazione**

### Step 5: Test

1. Logout from FixCity
2. Go to login page: `/auth/login`
3. Click **🔵 Accedi con Google**
4. Complete Google consent
5. Should redirect back and log in successfully

## Troubleshooting

### "redirect_uri_mismatch" Error

**Cause**: Redirect URL in Google Console doesn't match FixCity config.

**Solution**:
1. Check exact redirect URL in admin panel
2. Add to Google Console → Credentials → Authorized redirect URIs
3. Must match exactly (including https/http, trailing slash)

### "invalid_client" Error

**Cause**: Wrong Client ID or Secret.

**Solution**:
1. Re-copy from Google Console (no extra spaces)
2. Check Client ID ends with `.apps.googleusercontent.com`
3. Secret starts with `GOCSPX-`

### Provider Not Showing on Login Page

**Cause**: Provider not enabled or misconfigured.

**Solution**:
1. Check admin panel: Status must be **Attivo**
2. Both Client ID and Secret must be filled
3. Check `config/services.php` has provider config
4. Clear cache: `php artisan config:clear`
