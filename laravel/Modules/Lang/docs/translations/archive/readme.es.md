# 🚀 Bienvenido a Laraxot PTVX

> **La plataforma Laravel de nueva generación: modular, multi-tenant y type-safe para proyectos SaaS y empresariales ambiciosos.**

---

## ✨ ¿Por qué Laraxot PTVX?

- **Claridad:** Cada decisión arquitectónica está documentada con el "por qué" y el "qué".
- **Escalabilidad:** Diseñada para crecer con tus necesidades empresariales.
- **Mantenibilidad:** La type-safety y la modularidad reducen los bugs y la deuda técnica.
- **Experiencia de desarrollo:** El onboarding es rápido gracias a la documentación interconectada y viva.

---

## ✨ Características Principales

- **Arquitectura Modular:** Cada funcionalidad es un módulo autónomo para máxima escalabilidad y mantenibilidad.
- **Conformidad PHPStan Nivel 9:** Rigurosa type-safety y análisis estático en todo el código.
- **Multi-Tenancy:** Sirve a múltiples inquilinos con robusta aislación y flexibilidad.
- **Filament Admin:** Panel de administración moderno y extensible out-of-the-box.
- **Spatie Data Objects:** Transferencia y validación de datos con Spatie Laravel Data.
- **Event Sourcing Ready:** Soporte integrado para arquitecturas event-driven.
- **Internacionalización:** Soporte completo para proyectos multilingües.
- **Documentación Automatizada:** Cada módulo tiene su propia documentación viva y está conectado con la documentación global.

---

## 📚 Estructura del Proyecto

- `laravel/Modules/` — Toda la lógica de negocio, agrupada por módulo
- `laravel/Themes/` — Assets frontend tematizables
- `laravel/project_docs/` — Documentación core y de los módulos
- `docs/` — Documentación de proyecto, workflows y traducciones
