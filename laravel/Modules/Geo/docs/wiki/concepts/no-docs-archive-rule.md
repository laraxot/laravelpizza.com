# no docs archive rule

## Regola modulo Geo

Per nuova documentazione Geo non usare `docs/archive/`.

- usare `docs/wiki/` per conoscenza consolidata
- usare `docs/raw/` per input sorgente

Gli archivi storici restano sola lettura.
