## Sommario

il progetto utilizza convenzioni di naming standardizzate per:

1. **Campi Database**: `first_name`/`last_name` per i nomi delle persone
2. **Classi Base**: Pattern `XotBase{Type}` per le classi base nel modulo Xot
3. **Namespace**: Struttura corretta senza segmento `app` anche se i file sono fisicamente in quella directory
4. **Directory**: Case sensitivity rigorosa (sempre lowercase per directory standard)

Consulta la documentazione completa nel modulo Xot per maggiori dettagli, esempi di implementazione e best practice.
