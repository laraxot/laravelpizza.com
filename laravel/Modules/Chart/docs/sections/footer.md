# Footer Section

## Panoramica
Il footer è una sezione fondamentale che fornisce navigazione secondaria, informazioni di contatto e altri contenuti importanti. [Documentazione Completa](../laravel/Modules/Cms/docs/sections/footer-section.md)

## Struttura
```json
{
    "name": {
        "it": "Footer Principale",
        "en": "Main Footer"
    },
    "slug": "footer",
    "blocks": {
        "info": { ... },
        "links": { ... },
        "newsletter": { ... },
        "bottom": { ... }
    }
}
```

## Blocchi Principali
- **Info**: Logo, descrizione e contatti [Dettagli](../laravel/Modules/Cms/docs/blocks/footer.md#1-info-footer-footerinfoblock)
- **Links**: Menu di navigazione organizzati in colonne [Dettagli](../laravel/Modules/Cms/docs/blocks/footer.md#2-link-footer-footerlinksblock)
- **Social**: Link ai social media [Dettagli](../laravel/Modules/Cms/docs/blocks/footer.md#3-social-footer-footersocialblock)
- **Contatti**: Informazioni di contatto [Dettagli](../laravel/Modules/Cms/docs/blocks/footer.md#4-contatti-footer-footercontactblock)
- **Newsletter**: Form di iscrizione alla newsletter [Dettagli](../laravel/Modules/Cms/docs/blocks/footer.md#5-newsletter-footer-footernewsletterblock)
- **Link Rapidi**: Link rapidi con target configurabile [Dettagli](../laravel/Modules/Cms/docs/blocks/footer.md#6-link-rapidi-footer-footerquicklinksblock)

## Best Practices
- Struttura semantica e accessibile
- Design responsive e mobile-first
- Performance ottimizzata
- SEO-friendly
