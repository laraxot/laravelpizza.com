# Header Section

## Panoramica
La sezione header è un componente fondamentale che gestisce la navigazione principale del sito. [Documentazione Sezioni](../laravel/Themes/One/docs/sections.md#header)

## Struttura
```json
{
    "name": {
        "it": "Header Principale",
        "en": "Main Header"
    },
    "slug": "header",
    "blocks": {
        "logo": { ... },
        "navigation": { ... },
        "actions": { ... }
    }
}
```

## Blocchi Principali
- **Logo**: Gestione del logo del sito
- **Navigation**: Menu di navigazione principale
- **Actions**: Pulsanti e azioni utente
