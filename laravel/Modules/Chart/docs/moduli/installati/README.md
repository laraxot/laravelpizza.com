# Moduli Installati il progetto

## Filament
Admin panel e CRUD:
- Gestione risorse
- Form builder
- Table builder
- Dashboard widgets
- Personalizzazione tema

## Spatie Permission
Gestione ruoli e permessi:
- Permessi granulari
- Ruoli gerarchici
- Policy e gates
- Cache permessi
- Middleware auth

## Nwidart Modules
Sistema modulare:
- Gestione moduli
- Autoloading
- Migrazioni
- Seeders
- Providers

## Laraxot Modules
Gestione moduli avanzata:
- Integrazione Filament
- Multi-tenant
- Multilingua
- Media manager
- Activity log

## Laravel Sanctum
API authentication:
- Token management
- SPA authentication
- CSRF protection
- Rate limiting
- Session handling

## Laravel Telescope
Debug e monitoraggio:
- Request tracking
- Query logging
- Cache monitoring
- Queue jobs
- Mail tracking

## Laravel Horizon
Gestione code Redis:
- Queue monitoring
- Job metrics
- Failed jobs
- Rate limiting
- Process management

## Laravel Nova
Admin panel alternativo:
- Resource management
- Dashboard builder
- Action system
- Filtering
- Authorization

## Laravel Echo
WebSocket e real-time:
- Event broadcasting
- Presence channels
- Private channels
- Real-time updates
- Notifications

## Pusher
Notifiche push:
- Real-time events
- Channel management
- Webhook support
- Analytics
- Security
