# Indice Documentazione il progetto

Questa cartella contiene SOLO collegamenti bidirezionali alle documentazioni tecniche dei singoli moduli.
La documentazione approfondita, le scelte progettuali, roadmap, filosofia e dettagli tecnici sono suddivisi per modulo, secondo le seguenti regole:

- **Documentazione generale** → [Modules/Xot/docs](../../Modules/Xot/docs)
- **Documentazione frontend** → [Modules/Cms/docs](../../Modules/Cms/docs)
- **Componenti UI** → [Modules/UI/docs](../../Modules/UI/docs)
- **Altre documentazioni specifiche** → nella cartella `docs` del modulo pertinente (es: Activity, Reporting, ecc.)

## Come orientarsi

- Per informazioni generali, roadmap, filosofia, consultare la documentazione del modulo Xot.
- Per tutto ciò che riguarda il frontend, vedere la documentazione del modulo Cms.
- Per i componenti UI e linee guida di design, consultare la documentazione del modulo UI.
- Per tematiche specifiche (es. GDPR, Media, Tenant, ecc.), consultare la documentazione del relativo modulo.

> **Nota:** Se trovi documentazione generica in questa cartella, spostala nella documentazione del modulo Xot e lascia qui solo il collegamento.

### Collegamenti principali

- [Documentazione Xot (core, generica)](../../Modules/Xot/docs)
- [Documentazione Cms (frontend)](../../Modules/Cms/docs)
- [Documentazione UI (componenti UI)](../../Modules/UI/docs)

Per ogni approfondimento, segui i collegamenti all'interno delle rispettive cartelle `docs` dei moduli.
