# Dashboard Paziente

## Panoramica
La dashboard principale che offre un riepilogo delle informazioni e delle azioni principali per i pazienti registrati.

## Sezioni Principali

### 1. Riepilogo Appuntamenti
- Prossimo appuntamento in evidenza
- Stato delle prenotazioni in attesa
- Storico appuntamenti recenti
- Pulsante rapido per nuova prenotazione

### 2. Documenti
- Stato documenti caricati
- Avvisi per documenti mancanti o in scadenza
- Pulsante per caricare nuovi documenti

### 3. Notifiche
- Ultimi avvisi e promemoria
- Stato delle richieste in elaborazione
- Messaggi importanti dallo staff

### 4. Azioni Rapide
- Prenota visita
- Scarica certificati
- Modifica profilo
- Contatta assistenza

## Personalizzazione
- Tema chiaro/scuro
- Impostazioni notifiche
- Lingua preferita

## Accesso Mobile
- Interfaccia responsive
- Accesso rapido alle funzionalità principali
- Notifiche push
