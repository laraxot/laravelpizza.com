# Ricerca Studi Odontoiatrici

## Panoramica
Sistema di ricerca avanzata per trovare studi odontoiatrici convenzionati in base a vari criteri.

## Funzionalità

### 1. Filtri di Ricerca
- **Localizzazione**
  - Ricerca per indirizzo, città, CAP
  - Rilevamento posizione automatico
  - Mappa interattiva con raggio di ricerca

- **Disponibilità**
  - Filtro per data e fascia oraria
  - Disponibilità immediata
  - Giorni specifici della settimana

- **Specializzazioni**
  - Odontoiatria generale
  - Ortodonzia
  - Igiene dentale
  - Altri servizi specialistici

### 2. Risultati
- Elenco studi con informazioni essenziali
- Valutazioni e recensioni
- Distanza dalla posizione selezionata
- Indicazione posti disponibili

### 3. Ordinamento
- Per distanza
- Per valutazione
- Per disponibilità
- Per costo (se applicabile)
