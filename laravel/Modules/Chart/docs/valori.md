# Politica, Filosofia, Religione, Etica, Zen – Documentazione Generale

## Politica
La documentazione è uno spazio democratico: ogni voce può contribuire, ogni dubbio è legittimo.

## Filosofia
Documentare è condividere conoscenza: “Sapere è potere solo se condiviso”.

## Religione
Rispetto per tutte le fedi e nessuna. Il sapere tecnico è universale.

## Etica
La documentazione deve essere veritiera, accessibile, utile e mai fuorviante.

## Zen
Scrivere documentazione è meditazione: ordine, chiarezza, essenzialità. “Less is more.”
