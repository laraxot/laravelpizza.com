# Errori Comuni e Soluzioni

## Errori Ricorrenti
- Uso di nomi di progetto nei file `.md` o `.mdc`
- Mancanza di cross-link tra file di regole
- Modifica del codice senza aggiornare prima la documentazione
- Mancanza di checklist operative
- Mancanza di esempi generici e riutilizzabili

## Soluzioni
- Rimuovere ogni riferimento a nomi di progetto
- Aggiornare sempre la documentazione PRIMA del codice
- Aggiungere almeno 5 cross-link per ogni file `.md` e `.mdc`
- Usare la [checklist di ripartenza](checklist-di-ripartenza.md) prima di ogni intervento
- Consultare le regole in [Documentation Standards](../.windsurf/rules/documentation.mdc)
