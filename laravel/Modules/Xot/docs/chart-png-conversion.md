# Chart PNG Conversion Guide

## 📋 Overview

Guida dettagliata per la conversione di chart SVG in PNG utilizzando Imagick con ottimizzazioni per performance e qualità.

## 🖼️ Architettura Conversione

### 1. Servizio di Conversione

```php
<?php

namespace Modules\Xot\Services\Chart;

use Imagick;
use ImagickDraw;
use ImagickPixel;

class ChartPngConverter
{
    private array $defaultOptions = [
        'width' => 800,
        'height' => 400,
        'dpi' => 150,
        'quality' => 90,
        'compression' => 6,
        'background' => '#ffffff',
        'antialias' => true,
    ];

    public function convertSvgToPng(string $svg, array $options = []): string
    {
        $options = array_merge($this->defaultOptions, $options);

        try {
            // 1. Prepara SVG
            $preparedSvg = $this->prepareSvg($svg, $options);

            // 2. Crea Imagick instance
            $imagick = new Imagick();

            // 3. Configura Imagick
            $this->configureImagick($imagick, $options);

            // 4. Carica e processa SVG
            $this->loadAndProcessSvg($imagick, $preparedSvg, $options);

            // 5. Genera PNG
            $png = $this->generatePng($imagick, $options);

            // 6. Pulizia
            $imagick->clear();
            $imagick->destroy();

            return $png;

        } catch (\ImagickException $e) {
            throw new ChartConversionException("SVG to PNG conversion failed: " . $e->getMessage());
        }
    }

    private function prepareSvg(string $svg, array $options): string
    {
        // Aggiungi namespace se mancante
        if (!str_contains($svg, 'xmlns="http://www.w3.org/2000/svg"')) {
            $svg = str_replace('<svg', '<svg xmlns="http://www.w3.org/2000/svg"', $svg);
        }

        // Aggiungi viewBox se mancante
        if (!str_contains($svg, 'viewBox')) {
            $svg = str_replace('<svg', '<svg viewBox="0 0 ' . $options['width'] . ' ' . $options['height'] . '"', $svg);
        }

        // Pulizia whitespace eccessivo
        $svg = preg_replace('/\s+/', ' ', $svg);
        $svg = trim($svg);

        return $svg;
    }

    private function configureImagick(Imagick $imagick, array $options): void
    {
        // Imposta formato e risoluzione
        $imagick->setImageFormat('png');
        $imagick->setImageResolution($options['dpi'], $options['dpi']);

        // Imposta qualità e compressione
        $imagick->setOption('png:compression-level', $options['compression']);
        $imagick->setOption('png:compression-strategy', 1);
        $imagick->setOption('png:compression-filter', 5);

        // Imposta antialiasing
        if ($options['antialias']) {
            $imagick->setOption('png:bit-depth', 8);
        }

        // Imposta color space
        $imagick->setColorspace(Imagick::COLORSPACE_SRGB);
    }

    private function loadAndProcessSvg(Imagick $imagick, string $svg, array $options): void
    {
        // Carica SVG da blob
        $imagick->readImageBlob($svg);

        // Imposta dimensioni
        $imagick->resizeImage(
            $options['width'],
            $options['height'],
            Imagick::FILTER_LANCZOS,
            1
        );

        // Imposta background
        if ($options['background'] !== 'transparent') {
            $imagick->setImageBackgroundColor($options['background']);
        }

        // Unisci layer se necessario
        if ($imagick->getImageAlphaChannel()) {
            $imagick->setImageAlphaChannel(Imagick::ALPHACHANNEL_OPAQUE);
        }
    }

    private function generatePng(Imagick $imagick, array $options): string
    {
        // Imposta qualità
        $imagick->setImageCompressionQuality($options['quality']);

        // Genera blob PNG
        return $imagick->getImageBlob();
    }
}
```

### 2. Conversione Avanzata

```php
class AdvancedChartPngConverter extends ChartPngConverter
{
    public function convertSvgToPngWithWatermark(string $svg, array $options = []): string
    {
        $png = $this->convertSvgToPng($svg, $options);

        if ($options['watermark'] ?? false) {
            $png = $this->addWatermark($png, $options['watermark_text'] ?? 'Generated by PTVX');
        }

        return $png;
    }

    public function convertSvgToPngWithShadow(string $svg, array $options = []): string
    {
        $png = $this->convertSvgToPng($svg, $options);

        if ($options['shadow'] ?? false) {
            $png = $this->addShadow($png, $options['shadow_options'] ?? []);
        }

        return $png;
    }

    public function convertSvgToPngWithBorder(string $svg, array $options = []): string
    {
        $png = $this->convertSvgToPng($svg, $options);

        if ($options['border'] ?? false) {
            $png = $this->addBorder($png, $options['border_options'] ?? []);
        }

        return $png;
    }

    private function addWatermark(string $png, string $text): string
    {
        $imagick = new Imagick();
        $imagick->readImageBlob($png);

        $draw = new ImagickDraw();
        $draw->setFontSize(20);
        $draw->setFillColor('rgba(200, 200, 200, 0.5)');
        $draw->setGravity(Imagick::GRAVITY_SOUTHEAST);
        $draw->annotation(10, 10, $text);

        $imagick->drawImage($draw);

        $result = $imagick->getImageBlob();

        $imagick->clear();
        $imagick->destroy();

        return $result;
    }

    private function addShadow(string $png, array $options): string
    {
        $imagick = new Imagick();
        $imagick->readImageBlob($png);

        // Crea shadow
        $shadow = clone $imagick;
        $shadow->setImageBackgroundColor('rgba(0, 0, 0, 0.3)');
        $shadow->blurImage(5, 3);
        $shadow->compositeImage($imagick, Imagick::COMPOSITE_OVER, 5, 5);

        $result = $shadow->getImageBlob();

        $imagick->clear();
        $imagick->destroy();
        $shadow->clear();
        $shadow->destroy();

        return $result;
    }

    private function addBorder(string $png, array $options): string
    {
        $imagick = new Imagick();
        $imagick->readImageBlob($png);

        $width = $imagick->getImageWidth();
        $height = $imagick->getImageHeight();
        $borderWidth = $options['width'] ?? 2;
        $borderColor = $options['color'] ?? '#333333';

        // Crea bordo
        $border = new Imagick();
        $border->newImage($width, $borderWidth, $borderColor);

        // Unisci con immagine originale
        $border->compositeImage($imagick, Imagick::COMPOSITE_OVER, 0, 0);
        $border->compositeImage($imagick, Imagick::COMPOSITE_OVER, 0, $height - $borderWidth);

        $result = $border->getImageBlob();

        $imagick->clear();
        $imagick->destroy();
        $border->clear();
        $border->destroy();

        return $result;
    }
}
```

## 🚀 Utilizzo Avanzato

### 1. Batch Conversion

```php
class BatchChartConverter
{
    public function convertMultiple(array $charts, array $options = []): array
    {
        $results = [];

        foreach ($charts as $index => $chart) {
            try {
                $startTime = microtime(true);

                $png = $this->convertSvgToPng($chart['svg'], $chart['options'] ?? $options);

                $results[] = [
                    'index' => $index,
                    'success' => true,
                    'size' => strlen($png),
                    'time' => microtime(true) - $startTime,
                    'png' => $png,
                ];

            } catch (\Exception $e) {
                $results[] = [
                    'index' => $index,
                    'success' => false,
                    'error' => $e->getMessage(),
                ];
            }
        }

        return $results;
    }
}
```

### 2. Conversione con Cache

```php
class CachedChartConverter extends ChartPngConverter
{
    private function getCacheKey(string $svg, array $options): string
    {
        return 'chart_png_' . md5($svg . serialize($options));
    }

    public function convertSvgToPngCached(string $svg, array $options = []): string
    {
        $cacheKey = $this->getCacheKey($svg, $options);

        return Cache::remember($cacheKey, 3600, function () use ($svg, $options) {
            return $this->convertSvgToPng($svg, $options);
        });
    }

    public function clearCache(string $svg, array $options = []): void
    {
        $cacheKey = $this->getCacheKey($svg, $options);
        Cache::forget($cacheKey);
    }
}
```

### 3. Conversione Streaming

```php
class StreamingChartConverter extends ChartPngConverter
{
    public function convertSvgToPngStream(string $svg, array $options = []): \Generator
    {
        try {
            // Genera PNG
            $png = $this->convertSvgToPng($svg, $options);

            // Stream output
            $fp = fopen('php://output', 'wb');
            fwrite($fp, $png);
            fclose($fp);

            yield $png;

        } catch (\Exception $e) {
            throw new ChartConversionException("Streaming conversion failed: " . $e->getMessage());
        }
    }

    public function streamToResponse(string $svg, array $options = []): \Symfony\Component\HttpFoundation\StreamedResponse
    {
        return new \Symfony\Component\HttpFoundation\StreamedResponse(
            function () use ($svg, $options) {
                echo $this->convertSvgToPng($svg, $options);
            },
            200,
            [
                'Content-Type' => 'image/png',
                'Content-Disposition' => 'attachment; filename="chart.png"',
            ]
        );
    }
}
```

```php
<?php

namespace Modules\Xot\Tests\Unit\Services\Chart;

use Tests\TestCase;
use Modules\Xot\Services\Chart\ChartPngConverter;

class ChartPngConversionTest extends TestCase
{
    /** @test */
    public function it_converts_svg_to_png()
    {
```

### 2. Performance Tests

```php
class ChartConversionPerformanceTest extends TestCase
{
    /** @test */
    public function it_converts_large_charts_efficiently()
    {
        // Genera SVG complesso
        $svg = $this->generateComplexSvg(1000); // 1000 elementi

        $converter = new ChartPngConverter();

        $startTime = microtime(true);
        $png = $converter->convertSvgToPng($svg);
        $duration = microtime(true) - $startTime;

        // Dovrebbe convertire in meno di 5 secondi
        $this->assertLessThan(5.0, $duration);
        $this->assertGreaterThan(10000, strlen($png));
    }

    /** @test */
    public function it_handles_batch_conversion()
    {
        $converter = new \Modules\Xot\Services\Chart\BatchChartConverter();

        $charts = [];
        for ($i = 0; $i < 10; $i++) {
            $charts[] = [
                'svg' => $this->generateSimpleSvg(),
                'options' => ['width' => 400, 'height' => 200],
            ];
        }

        $startTime = microtime(true);
        $results = $converter->convertMultiple($charts);
        $duration = microtime(true) - $startTime;

        // Tutti dovrebbero avere successo
        foreach ($results as $result) {
            $this->assertTrue($result['success']);
        }

        // Batch conversion dovrebbe essere più veloce
        $this->assertLessThan(10.0, $duration);
    }

    private function generateComplexSvg(int $elements): string
    {
        $svg = '<svg width="800" height="600">';

        for ($i = 0; $i < $elements; $i++) {
            $x = rand(10, 790);
            $y = rand(10, 590);
            $color = sprintf('#%06x', rand(0, 0xFFFFFF));

            $svg .= "<circle cx=\"{$x}\" cy=\"{$y}\" r=\"5\" fill=\"{$color}\"/>";
        }

        $svg .= '</svg>';

        return $svg;
    }

    private function generateSimpleSvg(): string
    {
        return '<svg width="200" height="100"><rect x="10" y="10" width="180" height="80" fill="blue"/></svg>';
    }
}
```

## 📊 Ottimizzazione Performance

### 1. Riduzione Dimensioni

```php
class OptimizedChartConverter extends ChartPngConverter
{
    public function convertSvgToPngOptimized(string $svg, array $options = []): string
    {
        // Limita dimensioni massime
        $maxWidth = $options['max_width'] ?? 2000;
        $maxHeight = $options['max_height'] ?? 2000;

        $options['width'] = min($options['width'] ?? 800, $maxWidth);
        $options['height'] = min($options['height'] ?? 400, $maxHeight);

        // Usa compressione più alta per dimensioni grandi
        if ($options['width'] > 1000 || $options['height'] > 1000) {
            $options['compression'] = 9;
            $options['quality'] = 85;
        }

        return $this->convertSvgToPng($svg, $options);
    }
}
```

### 2. Cache Intelligente

```php
class SmartChartConverter extends ChartPngConverter
{
    private function shouldCache(string $svg, array $options): bool
    {
        // Non cache per chart piccoli
        $estimatedSize = strlen($svg) * 2; // Stima conversione SVG→PNG

        return $estimatedSize > 50000; // 50KB
    }

    public function convertSvgToPngSmart(string $svg, array $options = []): string
    {
        if ($this->shouldCache($svg, $options)) {
            return $this->convertSvgToPngCached($svg, $options);
        }

        return $this->convertSvgToPng($svg, $options);
    }
}
```

#### 3. SVG invalido

```php
// Valida SVG prima della conversione
private function validateSvg(string $svg): bool
{
    try {
        $dom = new \DOMDocument();
        $dom->loadXML($svg);

        return $dom->documentElement->tagName === 'svg';
    } catch (\Exception $e) {
        return false;
    }
}
```
