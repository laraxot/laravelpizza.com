## 🚀 NPM Commands

**⚠️ CRITICAL**: Always run NPM commands in theme directory!

```bash
# Navigate to theme directory
cd /var/www/_bases/base_laravelpizza/laravel/Themes/Meetup

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Watch for changes
npm run watch
```
