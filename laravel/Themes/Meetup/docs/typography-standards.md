# Typography Standards - Meetup Theme

## Core Principles

- **Readability**: Prioritize readability by choosing a clear, legible font and appropriate text sizes.
- **Hierarchy**: Establish a clear visual hierarchy for headings, subheadings, and body text to guide the user's eye and improve scannability.
- **Consistency**: Use a consistent set of font sizes, weights, and line heights throughout the application.

## Implementation Guidelines

- **Font Family**: Use a modern, sans-serif font like Inter, Lato, or Nunito for body text and headings.
- **Base Font Size**: A base font size of 16px is recommended for body text.
- **Scale**: Use a typographic scale (e.g., a major third scale) to determine heading sizes.
- **Line Height**: Use a line height of 1.5 for body text to improve readability.
- **Paragraphs**: Keep paragraphs relatively short to avoid overwhelming the user.

## Example Scale (based on Tailwind CSS)

- **`<h1>`**: `text-4xl` (36px), `font-bold`
- **`<h2>`**: `text-3xl` (30px), `font-bold`
- **`<h3>`**: `text-2xl` (24px), `font-bold`
- **`<h4>`**: `text-xl` (20px), `font-semibold`
- **Body**: `text-base` (16px), `font-normal`
- **Small Text**: `text-sm` (14px), `font-normal`
