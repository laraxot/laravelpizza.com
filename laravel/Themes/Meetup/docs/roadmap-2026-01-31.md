# Meetup Theme Roadmap - 2026-01-31

## 📊 Overall Progress: 65%

## 🎯 Theme Purpose
The Meetup theme is the frontend theme for Laravel Pizza, designed to be modern, responsive, and performant using Tailwind CSS v4, Vite, Laravel Folio for file-based routing, and Laravel Volt for declarative components.
