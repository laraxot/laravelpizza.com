# Laravel Pizza - Meetup Theme

## Overview
The Laravel Pizza Meetup Theme is a fully responsive, accessible, and modern HTML/CSS/JS theme designed for pizza delivery and ordering websites. It features a complete set of pages with enhanced functionality and accessibility.

## Pages Included
- **Home (index.html)** - Main landing page with hero section, features, and menu highlights
- **Menu (menu.html)** - Complete pizza menu with category filtering and cart integration
- **Cart (cart.html)** - Shopping cart with local storage persistence and quantity controls
- **About (about.html)** - Company information, values, and team
- **Contact (contact.html)** - Contact form with validation and location information

## Key Features

### Enhanced User Experience
- **Persistent Shopping Cart**: Uses localStorage to maintain cart items across sessions
- **Quantity Controls**: Increase/decrease item quantities directly in cart
- **Category Filtering**: Filter menu items by category
- **Smooth Animations**: Enhanced user experience with CSS transitions

### Accessibility Improvements
- **ARIA Labels**: Proper labels for screen readers
- **Keyboard Navigation**: Full keyboard support for all interactive elements
- **Skip Links**: Accessibility navigation for screen readers
- **Semantic HTML**: Proper use of HTML5 semantic elements
- **Focus Management**: Proper focus states and management

### SEO & Performance
- **Schema.org Markup**: Structured data for rich snippets
- **Meta Tags**: Proper SEO metadata
- **Lazy Loading**: Images loaded efficiently
- **Mobile Responsive**: Fully responsive design

### Modern Design
- **Tailwind CSS**: Utility-first CSS framework with custom theme
- **Pizza-themed Colors**: Red (primary), yellow (secondary), green (accent)
- **Consistent Typography**: Inter font family with proper hierarchy
- **Interactive Elements**: Hover effects and transitions

## Technical Implementation

### JavaScript Features
- **PizzaCart Class**: Advanced cart management with local storage persistence
- **Accessibility Utilities**: Screen reader announcements, focus management
- **Form Validation**: Enhanced validation with real-time error feedback
- **Category Filtering**: Dynamic content filtering

### CSS Features
- **Custom Tailwind Theme**: Pizza-themed color palette
- **Responsive Design**: Mobile-first approach with breakpoints
- **Component Classes**: Reusable UI components
- **Utility Classes**: Enhanced with custom utilities

## How to Use
1. Copy the HTML files to your web server
2. Ensure the CSS and JS files are in the correct paths (`./dist/css/app.css`, `./js/app.js`)
3. Customize the content, colors, and contact information as needed
4. Connect to your backend system for order processing

## Customization
- **Colors**: Modify the color palette in `css/app.css`
- **Content**: Update text, images, and contact information in HTML files
- **Functionality**: Extend JavaScript classes for additional features
- **Layout**: Use Tailwind utility classes for layout changes

## Browser Support
- Modern browsers with ES6 support
- Latest versions of Chrome, Firefox, Safari, Edge
- Mobile browsers (iOS, Android)
